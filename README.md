# ZEROGRA RA ALLIANCE — Landing Page

株式会社ZEROGRAが運営する「ZEROGRA RA ALLIANCE」の公式ランディングページ。

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- framer-motion / lucide-react
- Vercel

## Development

```bash
npm install
npm run dev
```

## Deploy

This project is configured for Vercel (Tokyo region). See `vercel.json`.
